# Природні Мандри — Frontend

Frontend-частина багатосторінкового сайту **Природні Мандри**.

Проєкт створений для популяризації еко-туризму в Україні, публікації історій
мандрівників, перегляду статей та взаємодії з контентом.

## Технології

- Next.js 15
- React
- TypeScript
- CSS Modules
- modern-normalize
- Axios
- TanStack Query
- Zustand
- Formik
- Yup
- Swiper
- React Hot Toast
- React Dropzone
- Prettier
- ESLint

## Як запустити проєкт

1. Клонувати репозиторій:

```bash
git clone https://github.com/alina-hryhorenko/pryrodni-mandry-front.git
```

2. Перейти в папку проєкту:

```bash
cd pryrodni-mandry-front
```

3. Встановити залежності:

```bash
npm install
```

4. Створити локальний `.env.local` на основі `.env.example`.

5. Запустити проєкт:

```bash
npm run dev
```

6. Відкрити:

```txt
http://localhost:3000
```

## Робота з гілками

- `main` — стабільна гілка.
- `develop` — основна робоча гілка.
- Для задач створюємо гілки від `develop`.

Приклади:

```txt
feature/header
feature/home-page
feature/stories-page
fix/header-navigation
```
